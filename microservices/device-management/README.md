# EMS AlgaSensors - Device Management

Este repositório contém o microsserviço de gerenciamento de dispositivos, parte do projeto EMS AlgaSensors.

## Sobre o Projeto

EMS AlgaSensors é um sistema baseado em microsserviços para monitoramento e gerenciamento de dispositivos sensores. Os componentes do projeto incluem:

- [ems-algasensors-temperature-processing](https://github.com/MAntonioST/ems-algasensors-temperature-processing) - Processamento de temperatura
- [ems-algasensors-temperature-monitoring](https://github.com/MAntonioST/ems-algasensors-temperature-monitoring) - Monitoramento de temperatura
- [ems-algasensors-device-management](https://github.com/MAntonioST/ems-algasensors-device-management) - Gerenciamento de dispositivos (este repositório)
- [ems-algasensors-meta](https://github.com/MAntonioST/ems-algasensors-meta) - Informações gerais sobre o projeto

## Arquitetura do Projeto

O gerenciamento de dispositivos permite a configuração e controle dos sensores no sistema.
